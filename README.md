# 🎸 VoiceLive Pro - Pédale Multi-Effets Vocale et Guitare

Application professionnelle de traitement audio en temps réel.

## 🚀 Installation

```bash
pip install -r requirements.txt
```

## 🎯 Test

```bash
cd voicelive_pro
python tests/test_architecture.py
```

## 📋 Fonctionnalités

✅ **Effets Vocaux**
- Doubling
- Harmony (à venir)
